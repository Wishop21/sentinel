# SENTINEL

Real-time global asset tracking and analytics platform.

Live tracking of aircraft (ADS-B via OpenSky), vessels (AIS via AISstream), 
and satellites (TLE via CelesTrak) with a WebGL globe, classification intelligence, 
and a growing analytical layer.

## Status
Under active development. Stages 1–2 complete.

## Stack
FastAPI · deck.gl · React · Skyfield · SQLite · Parquet
